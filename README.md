### Installation
#### Frontend
AngularJS Anwendung
=> in den Ordner `frontend` navigieren (`cd frontend`)
=> `npm install`
=> `yarn start`

#### Backend
Python Flask Anwendung
=> in den Ordner `backend` navigieren (`cd backend`)
=> `pip install -r requirements.txt` oder `python3 -m pip install -r requirements.txt`
=> `python3 server.py`
